Garden Eight
01 / Introduction

Wobble Scroll

Scroll down to experience the wobble effect. When you stop scrolling, content overshoots its target position and settles back with elastic motion. The effect comes from a lerp-based smooth scroll that chases a GSAP-tweened snap target.

02 / Mechanism

Lerp vs Tween

Every frame, the visual position lerps toward the target at ease 0.2. When scrolling stops, GSAP snaps the target. The visual position overshoots as it chases the suddenly-moved target, creating the wobble.

03 / Speed Distortion

Velocity Effects

Scroll velocity is tracked and can drive visual distortion — skew, scale, chromatic aberration. The speed value smooths through a 2-sample array with ease 0.025, creating organic acceleration and deceleration.

04 / Parameters

Tunable Values

ease: 0.2 · wheelRatio: 80 · snapTimeout: 100ms · snapDuration: 1s · speedEase: 0.1. These are the exact values extracted from the original kentatoshikura.com minified source.

05 / Infinite

Keep Scrolling

The vertical scroll is unbounded. Content wraps or extends. The wobble effect is always present — fast scrolls create dramatic overshoot, slow scrolls create gentle settling.
