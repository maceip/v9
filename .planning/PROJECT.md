# EdgeJS browser runtime (v9)

Node-shaped APIs in the browser: a WebAssembly-hosted runtime plus an N-API bridge that maps built-ins to browser capabilities.

**Product direction:** general-purpose **Node.js-in-tab** — run and test real Node apps, not a single vendor CLI. See `docs/NODEJS_IN_TAB_ROADMAP.md`.

**Release policy:** structured checks live in `.planning/phases/02-napi-bridge-hardening/02-03-PHASE-CLOSE-CHECKLIST.json` (`scripts/release-gate.mjs`).
